ABC 338 E - Chords Solution

前言：本人看到此题后开始分讨，没考虑全面，一直不过，最后讨出了 O(n) 做法，只需要 Dfs 的那种，但是 O(nlogn) 真的好想，好写 T^T

思路：

解法一：

考虑把每一条弦维护 x<y 后按左端点排序，那么两弦 i,j 有交当且仅当 a[i].y>=a[j].x，故我们想办法用数据结构维护一下前面出现过的弦的右端点即可。如果用单调栈维护，复杂度 O(n) ，如果是 set 或者是 线段树 之类的话 就需要 加一个 log 了。

解法二：( 建议比较闲的时候看，因为很没必要 T^T )

考虑如果一个图最终弦无交点，那么此时这个图一定是又若干个 环 和 若干条 互相平行的弦 组成 。

绿色圈起来的部分就是一个 环 。

我们考虑从 x 到 y 建边，然后 Dfs 一下，把所有在 环 里的边标记之后，对于所有未被标记的边，判断其是否都平行。

环的处理是 O(n) 的，弦的处理是 O(n) 的，总体下来是 O(n) 的。
